Roadmap

Getting evermore private.

v10 — Now

Ring-Sig Commitment Deposits

Standard CryptoNote deposits pioneered by DigitalNote & Conceal actually names the spent input (withdrawal) directly — ring signatures are mathematically impossible. Fuego's commitment I/O replaces that with indistinguishable outputs, so that both XFG deposits & withdrawals use ring-sigs with OSPEAD adaptive decoy selection. HEAT burns serve as permanent decoys.

v10.x — Next

Wallet Privacy + Dandelion++

2-output uniformity. IP-layer privacy via stem/fluff relay.

v11

Hidden Amounts — BP+ / CLSAG

Bulletproofs+ hide amounts. CLSAG proves balance in ring. Unified output type. Global ring pool. ~1.9 KB transactions.

v12

Triptych — Full-Chain Anonymity

Logarithmic ring proofs. Anonymity set expands from 9 decoys to the entire chain. Merkle membership over all outputs.
