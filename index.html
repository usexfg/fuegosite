Research & Development

v10 — new upgrades @ block 1M

Ring-Sig Commitment Deposits

The CryptoNote deposits originally pioneered by projects like DigitalNote or Conceal, directly names the spending (deposit) input on the withdrawal transaction— making ring signatures mathematically impossible. Fuego's commitment I/O model replaces this to create indistinguishable outputs, so that all XFG banking transactions (deposits & withdrawals) now use dynamic ring-signatures with max-achievable mixin & OSPEAD decoy selection (same as all Fuego txns)
Plus, coinbase outputs are binned by Power-of-10 to allow mixin with CD interest payments as decoys; effectively obfuscating "toxic" coinbase outputs to improve privacy for our miners.
